# ✍️ CNN Digit Recognition

A Convolutional Neural Network (CNN) based system to recognize handwritten digits (0–9) using the MNIST dataset, deployed with a Streamlit web interface.

---

## Tech Stack
- Python
- TensorFlow / Keras
- CNN
- Streamlit
